# План архитектуры многоязычного сайта

## Цели
1. ✅ Пользователь выбирает язык один раз
2. ✅ Показываются только статьи на выбранном языке
3. ✅ Выбор сохраняется между сессиями
4. ✅ Поддержка будущих языков (испанский)
5. ✅ SEO-friendly (URL параметры)

## Архитектура решения

### 1. Языковой переключатель
- Размещение: Header главной страницы
- Варианты: Выпадающий список или кнопки (RU | EN | ES)
- По умолчанию: Определяется по браузеру или RU

### 2. Хранение выбора
- **localStorage**: Сохранение выбора пользователя
- **URL параметр**: `?lang=ru` или `?lang=en` для SEO и возможности делиться ссылками
- **Приоритет**: URL параметр > localStorage > браузер > RU по умолчанию

### 3. Фильтрация контента
- На главной странице: Показывать только статьи выбранного языка
- В статьях: Кнопка "Back to Home" должна учитывать язык

### 4. Структура данных
```javascript
const articles = {
    ru: [
        { title: "WSDC 2025: Ивенты", url: "events_2025.html", ... }
    ],
    en: [
        { title: "WSDC 2025: Events", url: "events_2025_en.html", ... }
    ],
    es: [] // Будущие статьи
}
```

### 5. Навигация
- Кнопка "Back to Home" в статьях должна вести на `index.html?lang=ru` или `index.html?lang=en`
- Языковой переключатель должен быть доступен на всех страницах

## Реализация

### Шаг 1: Языковой переключатель в header
- Добавить dropdown или кнопки выбора языка
- Стилизовать в соответствии с дизайном сайта

### Шаг 2: JavaScript логика
- Функция определения языка (URL > localStorage > browser > default)
- Функция сохранения выбора
- Функция фильтрации статей
- Функция обновления навигации

### Шаг 3: Обновление статей
- Добавить языковой переключатель в header статей
- Обновить кнопку "Back to Home" для учета языка

### Шаг 4: Подготовка к испанскому
- Структура данных готова для добавления `es`
- Переключатель уже поддерживает 3 языка

## Преимущества
- ✅ Чистый UX - нет дубликатов статей
- ✅ SEO-friendly - URL параметры для индексации
- ✅ Масштабируемость - легко добавить новые языки
- ✅ Персистентность - выбор сохраняется
