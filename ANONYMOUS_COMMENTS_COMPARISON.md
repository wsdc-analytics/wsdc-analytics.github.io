# Сравнение систем комментариев без обязательной регистрации

## ⚠️ Обновление

**Выбрано решение: Cusdis**

Disqus был заменен на Cusdis, так как:
- Cusdis полностью бесплатен без скрытых платежей
- Нет рекламы
- Более легковесный
- Проще в настройке

---

## Варианты для GitHub Pages

### 1. Cusdis ✅ (Выбрано)

**Возможности:**
- ✅ **Полностью бесплатно** — без скрытых платежей
- ✅ **Без рекламы** — чистое решение
- ✅ Анонимные комментарии (имя + email)
- ✅ Комментарии через GitHub (опционально)
- ✅ Очень легковесный (быстрая загрузка)
- ✅ Open source (можно self-host)
- ✅ Простая интеграция

**Недостатки:**
- ⚠️ Менее популярен чем Disqus
- ⚠️ Нужна регистрация на cusdis.com для получения App ID

**Цена:** Бесплатно

**Подходит для:** Сайтов, где важна простота и отсутствие рекламы

---

### 1а. Disqus (Не выбрано — скрытые платежи)

**Возможности:**
- ✅ Анонимные комментарии (можно комментировать без регистрации)
- ✅ Комментарии через социальные сети (Facebook, Twitter, Google)
- ✅ Популярная система (многие пользователи уже имеют аккаунт)
- ✅ Хорошая модерация
- ✅ Мобильные приложения для модерации
- ✅ Аналитика комментариев

**Недостатки:**
- ❌ Показывает рекламу в бесплатном плане
- ❌ Платные тарифы могут быть скрыты или недоступны в некоторых регионах
- ❌ Тяжелее чем Giscus (больше нагрузка на страницу)
- ❌ Меньше контроля над данными

**Цена:** Бесплатно (с рекламой) или $10/месяц (без рекламы)

**Статус:** Не рекомендован — бесплатный план может быть недоступен или неочевиден

## Решение для WSDC Analytics

### ✅ Выбрано: Giscus + Cusdis

**Реализовано:**
- Giscus для пользователей с GitHub (по умолчанию)
- Cusdis для анонимных комментариев (переключение через табы)

**Почему Cusdis:**
- ✅ Полностью бесплатно без скрытых платежей
- ✅ Нет рекламы
- ✅ Поддерживает анонимные комментарии
- ✅ Легковесный и быстрый
- ✅ Простая интеграция

**Как это работает:**
- По умолчанию активна вкладка "GitHub (Giscus)"
- Пользователь может переключиться на "Cusdis (без регистрации)"
- Обе системы работают независимо
