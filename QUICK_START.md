# Быстрый старт

## Что готово

✅ Все файлы подготовлены в папке `/Users/ania/.cursor/wsdc-analytics-repo/`

## Что нужно сделать

### 1. Создать организацию на GitHub
- Название: **`wsdc-analytics`**
- План: Free

### 2. Создать репозиторий в организации
- Название: **`wsdc-analytics.github.io`** (обязательно точно так!)
- Public

### 3. Загрузить файлы

```bash
cd /Users/ania/.cursor/wsdc-analytics-repo
git init
git add .
git commit -m "Initial commit: WSDC Analytics articles"
git remote add origin https://github.com/wsdc-analytics/wsdc-analytics.github.io.git
git branch -M main
git push -u origin main
```

### 4. Включить GitHub Pages
- Settings → Pages → Source: `main` branch, `/ (root)` folder

### 5. Готово!
Сайт будет доступен через 1-2 минуты: **https://wsdc-analytics.github.io**
