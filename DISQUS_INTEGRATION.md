# Интеграция Disqus как альтернативы Giscus

## Статус интеграции

✅ **Готово:**
- Добавлены табы для переключения между Giscus и Disqus
- Структура HTML с обеими системами комментариев
- CSS стили для табов
- JavaScript для переключения между системами
- Информационные сообщения для каждой системы

⏳ **Требуется:**
- Регистрация в Disqus
- Получение Shortname
- Замена `YOUR-SHORTNAME` в коде

---

## Что нужно сделать

### Шаг 1: Регистрация в Disqus

1. Перейдите на https://disqus.com/
2. Нажмите **"Get Started"**
3. Зарегистрируйтесь (можно через Google/GitHub/Facebook или email)

### Шаг 2: Создать сайт

1. После регистрации нажмите **"I want to install Disqus on my site"**
2. Заполните форму:
   - **Website Name**: `wsdc-analytics` (или другое)
   - **Website URL**: `https://wsdc-analytics.github.io`
   - **Category**: `Education` или `Other`
   - **Language**: Для RU версии - `Russian`, для EN версии - `English`
3. Нажмите **"Create Site"**

### Шаг 3: Получить Shortname

После создания сайта Disqus покажет вам **Shortname** (уникальный идентификатор).

**Пример:** `wsdc-analytics` или `wsdcanalytics`

### Шаг 4: Настроить анонимные комментарии

1. В панели управления Disqus перейдите в **Settings** → **Moderation**
2. Включите опцию **"Allow guest commenting"**

### Шаг 5: Обновить код

Сообщите мне ваш Shortname, и я заменю `YOUR-SHORTNAME` в файлах:
- `events_2025.html`
- `events_2025_en.html`

**Или** вы можете заменить самостоятельно:
- Найдите в коде: `https://YOUR-SHORTNAME.disqus.com/embed.js`
- Замените `YOUR-SHORTNAME` на ваш Shortname

---

## Как это работает

1. **По умолчанию** активна вкладка "GitHub (Giscus)"
2. Пользователь может переключиться на "Disqus (без регистрации)"
3. При переключении показывается соответствующее информационное сообщение
4. Обе системы работают независимо друг от друга

---

## Преимущества решения

- ✅ Пользователи с GitHub могут использовать Giscus
- ✅ Пользователи без GitHub могут использовать Disqus
- ✅ Не нужно выбирать одну систему — обе доступны
- ✅ Переключение между системами в один клик
