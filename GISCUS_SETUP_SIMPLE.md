# Простая настройка Giscus

## Способ 1: Включить Discussions через создание (самый простой)

1. Откройте репозиторий: https://github.com/wsdc-analytics/wsdc-analytics.github.io
2. Нажмите на вкладку **"Discussions"** (слева в меню, рядом с "Pull requests", "Actions" и т.д.)
3. Если вкладки нет - нажмите на иконку с тремя точками "..." рядом с "Insights"
4. GitHub предложит включить Discussions - нажмите **"Set up discussions"**

## Способ 2: Через Settings

1. Откройте: https://github.com/wsdc-analytics/wsdc-analytics.github.io/settings
2. В левом меню найдите раздел **"General"**
3. Прокрутите вниз до раздела **"Features"** (или ищите "Discussions" в списке)
4. Найдите чекбокс "Discussions" и поставьте галочку ✅
5. Сохраните

## Способ 3: Прямая ссылка на Discussions

Попробуйте открыть напрямую:
https://github.com/wsdc-analytics/wsdc-analytics.github.io/discussions

GitHub должен предложить включить эту функцию.

---

## После включения Discussions:

1. Создайте категорию "General" (если нужно)
2. Установите Giscus App: https://github.com/apps/giscus
3. Получите ID: https://giscus.app/
